PWI by aarvay

A fast and a lightweight php-sdk for SASTRA's Parent Web Interface.

PWIpre-alpha is a fast and a lightweight PHP-SDK for SASTRA's Parent Web Interface. It can be used by developers to develop cool and useful applications. Synopsis: Set the two parameters(regno and password) and literally get anything that you need, in a neat JSON object.

Dependencies

PHP 5+, libcurl

Install

Unzip the folder and move the pwi directory to your project directory.
